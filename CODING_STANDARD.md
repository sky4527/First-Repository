# 项目代码规范（参照《阿里巴巴Java开发手册》）

1. **缩进**：使用4个空格，不使用Tab
2. **变量命名**：
    - 类名：UpperCamelCase（如 `KnapsackSolver`）
    - 方法名/变量名：lowerCamelCase（如 `calculateMaxValue`）
    - 常量：全大写+下划线（如 `MAX_CAPACITY`）
3. **每行最多字符数**：120
4. **函数最大行数**：80行（不含注释和空行）
5. **空行规则**：
    - 方法之间空1行
    - 不同逻辑块之间空1行
6. **注释规则**：
    - 类注释：说明类的功能和作者
    - 方法注释：说明功能、参数、返回值
    - 复杂逻辑：行注释说明思路
7. **操作符前后空格**：
    - 二元操作符前后加空格（如 `a + b`）
    - 一元操作符不加空格（如 `i++`）
8. **其他**：
    - 禁止魔法数字，使用常量代替
    - 所有覆写方法加 `@Override` 注解
